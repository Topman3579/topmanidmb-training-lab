# Game Design

## Loop

Hub → Briefing → 4 phases → Debrief → Progress

## Phases

1. **Evidence** — classify Observable / Derived / Inferred / Noise
2. **Timeline** — reorder events to match mock records
3. **Red Flag** — tap problematic lines in a simulated AI summary
4. **Decision** — pick the next investigative step

## Scoring

- Each phase: 0–100
- Total: average of completed phases
- Pass threshold: 70

## Principles

- Verification First
- Teaching-first feedback (explain why)
- No operational claims — training simulation only
